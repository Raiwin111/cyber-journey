# cyber-journey
lab ที่เอาไว้เรียนรู้เกี่ยวกับด้าน Cybersecurity IT ต่างๆ
เพื่อเข้าใจและเสริมทักษะการทํางานในด้านสายนี้ Fundamentals ที่ควรรู้และเข้าใจ logic พร้อมฝึก Soft Skill.

บันทึกการเรียนรู้ วันที่ 1 - Lab ติดตั้ง Windows
ติดตั้ง Windows 10 บน VM (Virtual Machine) เรียบร้อย

ทำความเข้าใจเรื่อง GPT และการแบ่ง Partition

ลองสำรวจ Task Manager กับ Services ต่างๆ

ลองจำลองปัญหา Network เล็กๆ น้อยๆ แล้วหัดแก้จนใช้งานได้
